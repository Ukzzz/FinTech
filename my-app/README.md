# FinTech Client (Frontend)

The frontend for the FinTech application, built with React and Tailwind CSS.

## ✨ Key Features
- **Dynamic Dashboards**: Real-time financial summaries and charts.
- **Expense Tracking**: intuitive interface for logging transactions.
- **Reporting System**: Export financial data to PDF and Excel formats.
- **Modern UI**: Polished, professional design with Framer Motion animations and Tailwind CSS.

## 🛠️ Scripts
- `npm start`: Runs the app in development mode.
- `npm run build`: Builds the app for production.
- `npm run tailwind`: Custom script to watch and compile Tailwind CSS output.

## 📦 Key Dependencies
- **`react-router-dom`**: client-side routing.
- **`recharts`**: data visualization library.
- **`lucide-react`**: high-quality icons.
- **`framer-motion`**: declarative animations.
- **`axios`**: HTTP client for API requests.
- **`xlsx` / `jspdf`**: file export utilities.

## 🎨 Styling
This project uses **Tailwind CSS**. Custom styles and configurations can be found in `tailwind.config.js`. 
Global styles and Tailwind directives are in `src/index.css`.
